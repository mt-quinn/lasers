## Big Lasers (client)

This is the playable web client for the laser game prototype.

### Run locally
From this folder:

```bash
npm install
npm run dev
```

Then open the URL Vite prints in the terminal.

### Controls
- Drag **anywhere**: moves the emitter along the bottom rail and aims the beam toward your finger.
- **Upgrades** button: pauses the run and opens the in-run upgrade menu.
- **Pause**: pauses/resumes without opening the menu.
